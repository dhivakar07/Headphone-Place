# 🎧 Headphone Place – MERN Stack Project

A full‑stack application built with **React (frontend)** and **Express + MongoDB (backend)** to manage audio products such as IEMs, Headphones, DACs/Amps, and Home Audio.  
This project demonstrates CRUD operations, file uploads, validation, and deployment on **Vercel**.

## 📌 Overview

The **Audio Store** project allows admins to add and manage audio products.  
It includes a product form with validation, image upload preview, and backend integration with MongoDB.  
The backend is designed to be **Vercel-ready**, exporting the Express app without `app.listen`.

---

## 🚀 Features

- **[Add Product](ca://s?q=Explain_Add_Product_feature)**: Upload product details with image preview.
- **[File Upload](ca://s?q=Explain_File_Upload_in_project)**: Supports JPEG/PNG cover images using `FormData` and `multer`.
- **[Validation](ca://s?q=Explain_Form_Validation)**: Ensures required fields are filled before submission.
- **[MongoDB Integration](ca://s?q=Explain_MongoDB_integration)**: Stores product data securely.
- **[Responsive UI](ca://s?q=Explain_Responsive_UI)**: Built with React components and modular CSS.
- **[Deployment Ready](ca://s?q=Explain_Vercel_deployment)**: Configured for Vercel hosting (frontend + backend).
